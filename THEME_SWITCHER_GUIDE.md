# Переключатель дизайна сайта

## Описание

На сайте теперь доступны два дизайна:

### 1. **Clinical Focus** (по умолчанию)
- Светлый белый фон
- Бирюзовые акценты (#48A9A6)
- Мятный вторичный фон (#EAF4F4)
- Современный, свежий вид

### 2. **Academic Paper** (Premium Academic / Old Money)
- Серо-кофейный фон (#F4F2F0)
- Строгая синяя палитра (#00075e)
- Белые карточки на сером фоне
- Академичный, сдержанный стиль

## Как использовать

1. **Кнопка переключения** находится в правом нижнем углу экрана
2. Нажмите на кнопку для переключения между дизайнами
3. Выбранная тема сохраняется в localStorage браузера

## Технические детали

### Файлы
- `src/components/ThemeToggle.tsx` - компонент кнопки переключения
- `src/app/globals.css` - стили обеих тем
- `src/app/layout.tsx` - интеграция кнопки в layout

### Как это работает
- При переключении добавляется/удаляется класс `academic-theme` на элемент `<body>`
- Все стили Academic темы имеют префикс `body.academic-theme`
- Выбор темы сохраняется в `localStorage` под ключом `theme`

### Цветовая палитра Academic Theme

```css
--bg-color: #F4F2F0;           /* Серо-кофейный фон */
--primary-color: #00075e;       /* Глубокий синий */
--accent-button: #00075e;       /* Синие кнопки */
--secondary-bg: #ffffff;        /* Белые карточки */
--text-secondary: #8C92AC;      /* Серый с синим подтоном */
```

## Особенности Academic дизайна

- **Фон сайта**: Светлый серо-кофейный (#F4F2F0) - как старая бумага
- **Карточки товаров**: Белые (#FFFFFF) на сером фоне - эффект "листа бумаги"
- **Кнопки**: Глубокий синий (#00075e) с заглавными буквами
- **Вторичные кнопки**: Прозрачные с синей рамкой
- **Удаление**: Серые кнопки вместо красных
- **Бейджи**: Белые с синей рамкой
- **Атмосфера**: Дорогой читальный зал, кабинет профессора

## Преимущества

- Яркие элементы товаров (свечи, стикеры, обложки книг) выделяются на спокойном фоне
- Отсутствие визуального шума
- Вызывает доверие к экспертности
- Выглядит дорого и сдержанно
