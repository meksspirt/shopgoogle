# Керівництво по кольоровій схемі "Clinical Focus"

## Основна палітра

```css
--primary-color: #00075e;      /* Глибокий темно-синій - текст і заголовки */
--accent-button: #48A9A6;      /* Спокійна бірюза - кнопки CTA */
--secondary-bg: #EAF4F4;       /* Світла м'ята - фон хедера */
--bg-color: #ffffff;           /* Чистий білий - основний фон */
```

## Застосування кольорів

### Хедер (Шапка)
- **Фон**: `#EAF4F4` (Світла м'ята)
- **Текст посилань**: `#00075e` (Темно-синій)
- **Телефон та іконки**: `#48A9A6` (Бірюза)

### Футер (Підвал)
- **Фон**: `#00075e` (Темно-синій)
- **Текст**: `#ffffff` (Білий)
- **Іконки**: `#48A9A6` (Бірюза для контрасту)
- **Копірайт**: `#000545` з opacity 0.7

### Кнопки

#### Головна кнопка (Додати в кошик, Оформити)
```css
background-color: #48A9A6;
color: #ffffff;
border-radius: 4px;
```
**Hover**: `#3D918E`

#### Вторинна кнопка (Редагувати, Детальніше)
```css
background-color: transparent;
border: 1px solid #00075e;
color: #00075e;
```
**Hover**: Фон `#00075e`, текст `#ffffff`

#### Кнопка видалення
```css
background-color: #E05D5D;
color: #ffffff;
```
**Hover**: `#d04545`

### Ціни
- **Основна ціна**: `#00075e` (Темно-синій)
- **Стара ціна (закреслена)**: `#9ca3af` (Сірий)

### Плашки статусу

#### Залишок товару (більше 10)
```css
background-color: rgba(72, 169, 166, 0.1);
color: #48A9A6;
```

#### Залишок товару (менше 10)
```css
background-color: #FFF4E5;
color: #D97706;
```

### Таблиці (Адмін-панель)
```css
th {
  color: #00075e;
  background-color: #EAF4F4;
  font-weight: 600;
}
```

## Приклади використання

### Кнопка в HTML
```html
<button class="btn btn-primary">Додати в кошик</button>
```

### Заголовок
```html
<h1 style="color: var(--primary-color)">Заголовок</h1>
```

### Ціна
```html
<span style="color: var(--primary-color); font-weight: 700;">1000 грн</span>
```

## Переваги цієї схеми

✅ **Професійний вигляд** - Синій колір асоціюється з довірою та стабільністю  
✅ **Чистота** - Білий фон створює відчуття простору  
✅ **Спокій** - М'ятні акценти додають свіжості без агресії  
✅ **Читабельність** - Високий контраст між текстом і фоном  
✅ **Єдність** - Всі елементи працюють в одній системі
